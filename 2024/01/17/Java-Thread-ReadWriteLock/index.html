Java Thread-ReadWriteLock

Author: 千越 Date: 2024-01-17  22:19:06

当有一个数据结构有很多线程进行读取操作，但是写操作频率相对较低时，使用 ReadWriteLock（读写锁）会得到优化。读写锁允许同时有很多读者，但是当有一个写锁时，读锁将阻塞，直到写锁被释放。读写锁能否提升性能需要试验。
