Java 中的数组是对象吗？

数组是对象，并且不是从某个类直接实例化来的，而是 JVM 创建的。以下《Java Language Specification》中的两句话谨记：

  1. In the Java programming language, arrays are objects (§4.3.1), are dynamically created, and may be assigned to variables of type Object (§4.3.2). All methods of class Object may be invoked on an array.（创建的数组的类的直接父类就是 Object）
  2. Every array has an associated Class object, shared with all other arrays with the same component type. [This] acts as if: the direct superclass of an array type is Object [and] every array type implements the interfaces Cloneable and java.io.Serializable.（容纳每一种类型的数组都有一个关联的 class 对象）
